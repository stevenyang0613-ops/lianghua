## 🏗️ 核心架构

```
转债数据
   ↓
[一票否决] → 信用/溢价/强赎/价格过滤
   ↓
[璇玑评分] ← 9因子加权 (双低0.29 质量0.19 HV0.19 ...)
[西部评分] ← 双低0.40 动量0.30 YTM0.30
   ↓
   ├─ 璇玑TOP40 ∩ 西部TOP30 = 共识池
   ↓
[共识内排序] → 选25只 + 3只缓冲
   ↓
[持仓期风控] → 追踪-5% / 组合-15% / 周频调仓
```
